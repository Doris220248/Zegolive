Zego Live Streaming - Android

This Android project uses the Zego SDK to enable real-time live streaming and video calls.

Features:
Live video and audio streaming
Join and leave live rooms
Camera and microphone control
Low latency streaming

Requirements:
Android Studio Ladybug or newer
Android SDK 24 or higher
Zego AppID and AppSign from your Zego developer console

Setup:
Open the project in Android Studio
Add your Zego AppID and AppSign in the config file inside app/src/main/java/
Sync Gradle and build the project
Add  Readme
